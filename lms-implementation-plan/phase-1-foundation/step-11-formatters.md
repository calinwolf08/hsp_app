# Step 11: Formatter Utils

## Objective
Implement pure functions for formatting dates and display text.

## Files to Create
- `src/lib/features/lms/shared/utils/formatters.ts`

## Functions to Implement
- `formatDate(isoString)` - Format ISO date to readable string
- `formatRelativeTime(isoString)` - "2 days ago" format
- `formatDuration(seconds)` - "1h 30m" format
- `formatPercentage(value)` - "75%" format
- `formatEnrollmentSource(source)` - Readable source name
- `formatActivityType(type)` - Readable activity type
